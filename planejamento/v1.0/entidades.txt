Estoque
    - produto -> Produto
    - quantidadeDisponivel -> Integer
    - custo -> BigDecimal
    - venda -> BigDecimal
    - status -> Status
    - observacoes<> -> String

Produto
    - categoria -> Categoria
    - código -> String
    - marca -> Marca
    - detalhes<> -> String
        - opcional, um campo com observações sobre algum detalhe do produto

Pastilha (É UM produto)
    - conversoes<> -> Conversao
    - material -> Material ou String
        - inicialmente, podem ser null [ENUM?]
    - comprimento -> inicialmente, podem ser null
    - altura -> inicialmente, podem ser null
    - espessura -> inicialmente, podem ser null

Marca [Enum]
    - ECOPADS
    - FRASLE
    - COBREQ
    - BLUE_FRICTION

Categoria [Enum]
    - PASTILHA

Conversao
    - codigo -> String
    - marca -> String
    - aplicacoes<> -> Aplicacao
    - similares<> -> Conversao
    - observacoes<> -> String

Aplicacao
    - montadora -> String
        - Ex: Honda
    - veículo -> String
        - Ex: Civic
    - anoInicial -> Integer
    - anoFinal -> Integer
    - lado -> Lado
    - detalhes -> String
        - Ex: 1.6i 16V VTEC
        - Definir padrão
        - Considerar atributo 'motor'

Lado [Enum]
    - DT
    - TR
    - LE
    - LD
    - DT_TR
    - DT_LE
    - DT_LD
    - TR_LE
    - TR_LD
    - LE_LD

Status [Enum]
    - PEDIR
    - NAO_PEDIR

Movimentacao - achar um nome melhor
    - tipo -> TipoMovimentacao
    - data -> Date 
        - ou outro tipo usado para datas
    - parte -> String [cliente/fornecedor]
        - versões futuras: dividir entre Cliente e Fornecedor, usar herança nesses casos, destacando a diferença em cada classe individualmente
    - item<ItemMovimentacao> -> ItemMovimentacao
    - total -> BigDecimal

ItemMovimentacao
    - quantidade -> Integer
    - descrição -> String
    - valorUnitário -> BigDecimal
    - lado -> Lado
        - Automatico para produtos do estoque
    - subTotal -> BigDecimal

ItemMovimentacaoVenda (EXTENDS ItemMovimentacao)
    - produto -> Produto

ItemMovimentacaoServico (EXTENDS ItemMovimentacao)
    - servico -> String
        - Qual o serviço que está sendo realizado, usinagens de disco de freio, de cilindros de roda, de pinça, etc.

TipoMovimentacao [Enum]
    - ENTRADA
    - SAIDA
